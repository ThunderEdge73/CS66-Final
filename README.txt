HOW TO PLAY: Run the main.py file in any Python editor and type either "new" or "load" depending on whether you want to start a new save or load save data from the same directory as main.py. When starting a new save, you can name your hero character or leave the entry field blank to have the game choose a random name for you.

The help command will bring up a list of other commands to use and serves as a status screen for the hero's main stats. The fight command will initiate a fight with a random enemy from the pool as we do not yet have logic for assigning locations specific enemy pools.
